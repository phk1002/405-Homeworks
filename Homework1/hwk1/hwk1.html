Vocab Definitions

  • HTTP
    • Definition:
      • HTTP stands for HyperText Transfer Protocol. HTTP is a protocol that lets any sort of media to be transfered betweeen browsers and servers.
    • Link to Official Docs
  • HTML
    • Definition:
      • HTML stands for HyperText Markup Language. HTML is a language used to develop a webpage.
    • Link to Official Docs
  • Element
    • Definition:
      • An element is a part of language used to develop a webpage. An element consists of attributes in the opening tag, text content and a closing tag
    • Link to Official Docs
  • Tag
    • Definition:
      • A tag is made of the greater than/less than brackets, which are essential in creating elements.
    • Link to Official Docs
  • Block(CSS)
    • Definition:
      • A block is a type of HTML element that lets a new line to be formed underneath the previous element and above the next element.
    • Link to Official Docs
  • Doctype
    • Definition:
      • Doctype is written as <!DOCTYPE html>. It is a required preamlbe that is always found at the beginning of all documents. THe Doctype is written to make sure the web browser can be displayed a specific way according to the code.
    • Link to Official Docs
  • Head
    • Definition:
      • The head is the part of HTML that describes the document. For example, the author of the document.
    • Link to Official Docs
  • Attribute
    • Definition:
      • An attribute extends a tag, which changes the function of the tag. An attribute is written as name=value to let the attribute to be understood.
    • Link to Official Docs
  • Boolean
    • Definition:
      • A boolean is a logical data that only has the values of true/false or to either repeat/display a particular statement.
    • Link to Official Docs
  • DOM
    • Definition:
      • DOM stands for Document Object Model, which is a model of all the elements, test, string, comments, etc. used in a document. The DOM is displayed as a node tree in which each of the parts of the document are displayed as the nodes of the tree.
    • Link to Official Docs
  • CSS
    • Definition:
      • CSS stands for Cascading Style Sheet, which is a declaritive language which determines how a webpage will look like in a browser. The CSS styles are associated with other elements to display them in a specific way on a browser.
    • Link to Official Docs
  • Property(CSS)
    • Definition:
      • A CSS property is a characteristic that is associated with a value that is assigned to an aspect of how a browser should display the element.
    • Link to Official Docs
  • Selector(CSS)
    • Definition:
      • A CSS Selector is the part of a CSS Rule that describes specific elements in a document to a rule. When the elements are read, the rule that was matched to the element will be applied to them.
    • Link to Official Docs
